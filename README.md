О чем игра:
Это мобильная 3D-игра в жанре шутера, где ты защищаешь планету от нашествия летающих тарелок. Игрок управляет турелью и стреляет по НЛО, которые появляются из порталов в небе. Чем дольше играешь - тем сложнее становится.

Основные механики:
- Бесконечные волны НЛО, которые появляются из случайных порталов
- Мобильное управление
- Система сложности - со временем НЛО становятся быстрее и живучее
- Поражение, если пропустишь слишком много тарелок
- Паузы между волнами каждые 15 секунд
- Визуальные эффекты взрывов и анимации

Система врагов:
- НЛО летают по сложным траекториям (меняют направление)
- У каждого своя скорость, здоровье, поведение
- Случайное появление из разных порталов
Управление на телефоне:
-Левая часть экрана - джостик для движения
-Правая часть экрана - управление камерой
-Отдельные кнопки для прыжка и стрельбы

Игра построена по правилам хорошего программирования:

Разделение ответственности:
- Каждый скрипт отвечает за свою задачу
- Управление игроком отдельно от камеры
- Логика игры отдельно от врагов
Гибкая система событий:
- Компоненты общаются через события, а не прямые ссылки
- Легко добавлять новую функциональность
-Код не превращается в "спагетти"
Простота расширения:
- Легко добавить новые типы врагов
- Можно создать систему улучшений оружия
- Просто добавить боссов или новые уровни

Из минусов можно было бы побольше ООП использовать, добавить функции смены оружия, разные дополнения для игры под слоем абстракции, UI меню и управление по лучше но я очень торопился. Игра сделана за 2 дня.
